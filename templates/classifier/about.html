Tentang Autism Classifier

Aplikasi berbasis kecerdasan buatan untuk membantu deteksi dini ciri-ciri autisme melalui analisis gambar wajah.

Apa itu Autism Classifier?

Autism Classifier adalah aplikasi web yang menggunakan teknologi Deep Learning untuk menganalisis gambar wajah dan mendeteksi kemungkinan ciri-ciri autisme. Aplikasi ini dikembangkan sebagai alat bantu untuk screening awal, bukan sebagai pengganti diagnosis medis profesional.

Dengan memanfaatkan model Convolutional Neural Network (CNN) yang telah dilatih dengan dataset gambar wajah, aplikasi ini dapat memberikan prediksi dengan tingkat akurasi yang cukup tinggi.

Teknologi yang Digunakan

  • MTCNN (Multi-task Cascaded Convolutional Networks)

    Untuk deteksi wajah, landmark detection, dan face alignment.

  • Convolutional Neural Network (CNN)

    Model deep learning untuk klasifikasi gambar wajah.

  • TensorFlow & Keras

    Framework machine learning untuk training dan inference.

  • Django Web Framework

    Backend framework dengan clean architecture pattern.

Arsitektur Model

Struktur CNN yang digunakan untuk klasifikasi

Input Layer
224 × 224 × 3
Conv2D Layers
24→32→64 filters
Output
Sigmoid (0-1)
Detail Arsitektur:
Layer Type Parameters
1 Input shape=(224, 224, 3)
2 Conv2D + ReLU 24 filters, 3×3 kernel
3 MaxPooling2D 2×2 pool
4 Conv2D + ReLU 32 filters, 2×2 kernel
5 MaxPooling2D 2×2 pool
6 Conv2D + ReLU 64 filters, 1×1 kernel
7 MaxPooling2D 2×2 pool
8 Flatten -
9 Dense + Sigmoid 1 unit (binary output)
